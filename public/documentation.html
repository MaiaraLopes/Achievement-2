Movies API

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Content

Request URL HTTP Method Response
Get a list of movies /movies GET A JSON object holding a list with all movies in the catalog.
Get data about a specific movie /movies/:title GET A JSON object holding data about the movie specified, including an ID.
[
id: 1,
title: 'Coco',
studio: 'Disney',
genre: 'animation',
director: 'Lee Unkrich'
]
Get data about a specific director /directors/:director GET A JSON object holding data about the director specified.
[
name: 'Lee Unkrich',
nationality: 'United States of America',
movies: 'Toy Story 3', 'Coco',
]
Add new user /users POST A JSON object holding data about the new user that was added, including an ID.
[
id: 1,
name: 'Walter Disney',
date of birth: '5 Dec 1901',
country: 'United States of America',
]
Update existing user /users/:username PUT An alert indicating the user's profile has been updated.
Delete existing user /users/:username DELETE An alert indicating the user's profile has been deleted.
Get a list of user's movies /users/:username/movies GET A JSON object holding the list of the user's favorite movies.
Add a movide to user's list /users/:username/movies/:title POST A JSON object holding data about the movie added to the user's list
[
id: 1,
title: 'Coco',
studio: 'Disney',
genre: 'animation',
director: 'Lee Unkrich'
]
Delete a movie from the user's list users/:username/movies/:title DELETE An alert indicating the movie has been removed from the user's list.

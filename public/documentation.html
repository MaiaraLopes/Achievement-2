Movies API

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Content

Description URL HTTP Method Request Body Response
Get a list of movies /movies GET None A JSON object holding a list with all movies in the catalog.
Get data about a specific movie /movies/:title GET None A JSON object holding data about the movie specified, including an ID. 
{
id: 1,
title: 'Coco', 
studio: 'Disney', 
genre: 'animation',
director: 'Lee Unkrich' 
}
Get the genre of a specific movie /genres/:title GET None A JSON object holding data about the genre of the specific movie, including an id. 
{
id: 1,
title: 'Coco',
description: 'Animation is a method in which figures are manipulated to appear as moving images.'
}
Get data about a specific director /directors/:director GET None A JSON object holding data about the director specified. 
{
name: 'Lee Unkrich',
nationality: 'United States of America',
movies: 'Toy Story 3', 'Coco',
}
Add new user /users POST A JSON object holding data about the new user to be added. 
{
name: 'Walter Disney',
date of birth: '5 Dec 1901',
country: 'United States of America',
}
A JSON object holding data about the new user that was added, including an ID. 
{
id: 1,
name: 'Walter Disney',
date of birth: '5 Dec 1901',
country: 'United States of America',
}
Update existing user /users/:username PUT A JSON object holding the data about the information to be updated on the user's profile. 
{
name: 'Walter Elias Disney',
}
An alert indicating the user's profile has been updated.
Delete existing user /users/:username DELETE None An alert indicating the user's profile has been deleted.
Get a list of user's movies /users/:username/movies GET None A JSON array holding the list of the user's favorite movies.
[
{
id: 1,
title: 'Coco
studio: 'Disney',
genre: 'animation',
director: 'Lee Unkrich'
},

...

]
Add a movie to user's list /users/:username/movies/:title POST A JSON object holding data about the movie to be added to the user's list. 
{
title: 'Coco', 
studio: 'Disney',
genre: 'animation',
director: 'Lee Unkrich' 
}
A JSON object holding data about the movie added to the user's list. 
{
id: 1, 
title: 'Coco', 
studio: 'Disney',
genre: 'animation',
director: 'Lee Unkrich' 
}
Delete a movie from the user's list users/:username/movies/:title DELETE None An alert indicating the movie has been removed from the user's list.
